Grid Upper Tools

This toolbar features the following main elements:

  • A panel for zooming in/out of the grid
  • A panel for undo/redo'ing changes made on the grid
  • A panel for setting the grid manipulating mode
  • A panel for selecting the state
  • A panel for displaying the information about the currently hovered cell

The Zoom Panel is formed of the following elements:

  1. Zoom In button
    2. To zoom in on the grid
  2. Zoom Out button
    3. To zoom out of the grid
  3. Scale Information label
    4. To view the current zooming scale

The Undo/Redo Panel is formed of the following elements:

  1. Undo button
    2. To undo the last change made on the grid
  2. Redo button
    3. To redo the last chaneg made on the grid

The Mode Selection Panel is formed of the following elements:

  1. Draw Mode button
    2. To enable drawing on the grid
  2. Pick Mode button
    3. To enable selecting states from the grid
  3. Move Mode button
    4. To enable panning around on the grid

The State Selection Panel is formed of the following elements:

  1. Previous State button
    2. To select the previous state on the grid.
  2. Next State button
    3. To select the next state on the grid.
  3. Current State panel
    4. Displays a box colored with the corresponding state color and a label with the name of that state.

The Cell Information Panel is a pair of labels displaying the (X,Y) coordinates and the state of the currently hovered cell.
