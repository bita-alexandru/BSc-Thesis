Rules Panel

The Rules Panel is formed by the following elements:

  • A button for launching the Rules Editor
  • A search bar for searching into the list of rules
  • A list which displays all the available rules

The states list features a context menu that pops up when the right click button is triggered onto a rules.
The following options are available:

  • Select
    Select the rules
  • Go To
    Launch the States Editor and goes to the line with the rules
  • Change Color
    Change the color of the rules
  • Erase
    Erase all the cells of the rules
  • Delete
    Delete the rules and the cells of that rules
